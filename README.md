## 📌 Описание проекта

Образовательный **цифровой двойник астрофизического явления** — падения астероида на планету с атмосферой.  
Симуляция разработана в **Unity (URP)** и наглядно демонстрирует законы небесной механики: гравитационное притяжение по закону Ньютона, атмосферное торможение, изменение траектории и удар о поверхность.

---

## 🎬 Сценарий симуляции

1. **Старт** — астероид вылетает из заданной точки с начальной скоростью по касательной к планете (v₀ = 60 ед/с по оси Z)
2. **Первое сближение** — пролетает через атмосферу, сгорая и теряя скорость (BurnEffect)
3. **Разворот** — гравитация замедляет астероид и разворачивает его обратно
4. **Повторный вход** — второй проход через атмосферу, дополнительное торможение
5. **Падение** — астероид падает на поверхность планеты под действием гравитации
6. **Удар** — образуется кратер с лавой, поднимаются облака дыма

## ⚙️ Реализованные физические явления

| Явление | Реализация |
|---|---|
| Гравитационное притяжение | Закон Ньютона F = G·m₁·m₂/r², `GravityBody.cs` |
| Атмосферное торможение | Триггер `Atmosphere` + гасящая сила к Rigidbody |
| Сгорание в атмосфере | `ParticleSystem` (BurnEffect) |
| Траектория полёта | `TrailRenderer` на астероиде |
| Вращение планеты | `PlanetRotation.cs`, rotationSpeed = 5 |
| Образование кратера | `craterPrefab` в `AsteroidEffects.cs` |
| Магнитное поле | `MagneticZone` (tag: MagneticField) + `MagneticSensor.cs` |
| Дым и обломки | `Particle System` при ударе |

## 📜 Пользовательские скрипты C\#

### `GravityBody.cs`
Реализует закон всемирного тяготения. Каждый физический кадр (`FixedUpdate`) вычисляет вектор гравитационной силы от всех тел с этим компонентом и применяет его через `Rigidbody.AddForce()`.  
**Параметры:** `gravityConstant = 2`, `initialVelocity = (0, 0, 60)` для астероида, `(0, 0, 0)` для планеты.

### `AsteroidEffects.cs`
Управляет визуальными эффектами при взаимодействии астероида:
- Активирует `burnParticles` при входе в атмосферу
- При столкновении с поверхностью порождает `craterPrefab` (кратер + лава)

### `PlanetRotation.cs`
Вращает планету вокруг своей оси (`rotationSpeed = 5`) в каждом кадре `Update()`.

### `MagneticSensor.cs`
Детектирует вхождение объекта в `MagneticZone` (tag: `MagneticField`) и активирует `sensorParticles`.
